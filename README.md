# Customer Churn Prediction

## Overview
This project predicts whether a telecom customer is likely to churn using machine learning models. It includes data preprocessing, exploratory analysis, model building, and business insights.

## Models Used
- Logistic Regression
- Random Forest

## Key Results
Logistic Regression performed better in terms of accuracy, F1-score, and ROC-AUC.

## Business Insights
- High monthly charges increase churn risk
- New customers are more likely to churn
- Month-to-month contracts have higher churn rates

## Tools Used
- Python
- pandas
- numpy
- matplotlib
- scikit-learn
